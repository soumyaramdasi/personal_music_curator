# Personal Music Curator

Personal Music Curator is a command-line music recommendation application developed in Java. It provides personalized song recommendations by asking users about their musical preferences, including genre, language, artist, and preferred decade.

The project was developed to strengthen object-oriented programming concepts while exploring how recommendation systems can be built using simple filtering techniques.

---

## Features

- Recommend songs based on user preferences
- Filter recommendations by genre
- Filter recommendations by language
- Recommend songs from specific decades
- Interactive command-line interface

---

## Technologies Used

- Java
- Object-Oriented Programming
- ArrayLists
- Command-Line Interface (CLI)
